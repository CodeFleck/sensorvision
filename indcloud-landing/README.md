# Industrial Cloud Landing Page

This repository hosts the landing page for [Industrial Cloud](https://indcloud.io) - The IoT Platform That Scales With You.

## Deployment

This site is automatically deployed via GitHub Pages when changes are pushed to the `main` branch.

### Custom Domain Setup

1. The `CNAME` file contains the custom domain: `indcloud.io`
2. DNS records should point to GitHub Pages:
   - A records: `185.199.108.153`, `185.199.109.153`, `185.199.110.153`, `185.199.111.153`
   - CNAME record: `indcloud.github.io`

## Local Development

Simply open `index.html` in your browser - no build step required.
